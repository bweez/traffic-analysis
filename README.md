# traffic-analysis
Explore the WSDot traffic APIs and Annual reports

# Goals:

Automobile traffic has many costs from mental to environmental.  This project is meant to be an exploration of how to analyze traffic and improve it.

Play will [Deno.js](https://deno.land/), the new JS and TS runtime built on Rust.

# Resources:

* WSDOT (Washington State Department of Transportation) [API](https://www.wsdot.wa.gov/traffic/api/)
* WSDOT annual [traffic report pdfs](http://wsldocs.sos.wa.gov/library/docs/dot/annualtrafficrpt/annualtrafficrpt_home.aspx) and [other data sources](https://www.wsdot.wa.gov/mapsdata.htm)
* Google maps API
